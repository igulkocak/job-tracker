# 🚀 JobTrackr — AI-Powered Job Application Tracker

A fullstack web application to track job applications, manage progress, and optimize job search workflow.

## ✨ Features

- 🔐 Authentication (JWT)
- 📊 Dashboard with statistics
- 🔍 Search & filtering
- 🗂️ Job status management (Applied, Interview, Offer, Rejected)
- 📝 Add & edit job applications
- ⚡ Fast and responsive UI

## 🧠 Upcoming Features

- AI-powered CV suggestions
- Cover letter generation

## 🛠️ Tech Stack

### Frontend
- React
- CSS Modules

### Backend
- Node.js
- Express

### Database
- MongoDB
